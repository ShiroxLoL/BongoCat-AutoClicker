# BongoCat AutoClicker

A simple Python auto-clicker for *Bongo Cat*.  
Made by [ShiroxLoL](https://github.com/ShiroxLoL).  
For educational or personal use only.

---

## 🧩 Requirements
- [Python 3.x](https://www.python.org/downloads/)
- Install dependencies:
```bash
pip install -r requirements.txt
````

---

## ▶️ Usage

```bash
python autoclicker.py
```

* **Press `p`** → Start/Stop clicking.
* **Press `q`** → Exit.

---

## ⚙️ Settings

Edit these in `autoclicker.py` if needed:

```python
min_interval = 0.01
max_interval = 0.02
hold_time = 0.02
```
